# Model Registry

Use this file when it helps to think in bundles instead of isolated models.

## Core Bundle

This bundle often carries the most freight:

- inversion
- incentives
- base rates
- opportunity cost

## Mixing Rule

Better question than “Which single archetype is this?”:

- what is the core decision?
- what is the main distortion?
- what is the main source of downside?

Mix only the bundles that clarify those questions.

## Reminder

- mixed cases are normal
- one dominant bundle plus one supporting bundle is usually enough
